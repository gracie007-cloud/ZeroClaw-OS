## Get Started

### Docker (Fastest)
```bash
docker pull blueshirtprogrammer/zeroclaw
docker run -p 50001:80 blueshirtprogrammer/zeroclaw
```

### From Source
```bash
git clone https://github.com/blueshirtprogrammer/zeroclaw.git
cd zeroclaw
./run_ui.py
```

### Cloud Deploy
Deploy to **Render** (free) or any cloud provider. See [DEPLOY.md](DEPLOY.md) for full guide.
